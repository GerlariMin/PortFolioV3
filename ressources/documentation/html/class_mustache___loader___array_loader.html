Detailed Description

Mustache Template array Loader implementation.

An ArrayLoader instance loads Mustache Template source by name from an initial array: 

$loader = new ArrayLoader(
    'foo' => '{{ bar }}',
    'baz' => 'Hey {{ qux }}!'
);

$tpl = $loader->load('foo'); // '{{ bar }}'

The ArrayLoader is used internally as a partials loader by Mustache_Engine instance when an array of partials is set. It can also be used as a quick-and-dirty Template loader.
